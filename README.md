# mio2's setup environments for Linux Branch

## PreRequirements
- zsh
- curl
- wget

## Installaion
```
chmod +x setup.sh
./setup.sh
```

## Contains
- zsh
    - prezto
    - zsh-completions
    - zsh-autosuggestions
- byobu
- linuxbrew

- wsl2 display support (WSL2 only)

## addon
 - ~/.config/shell

## WSL
- setup ssh agent on [wsl2-ssh-agent](https://github.com/mame/wsl2-ssh-agent)
